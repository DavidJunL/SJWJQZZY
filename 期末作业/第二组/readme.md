# 摘要
通过爬虫获取成都信息工程大学481篇论文，分析整理出这些论文R的名字，作者信息，发表刊物，发表时间，分类，关键词存入一个csv文件中，
使用R语言进行数据分析该学校的所有科研成果，从不同的角度，例如从年份，刊物分布，关键词分析我校主攻的领域等
将这些功能集合在一起，生成一份html的数据报（markdown）。

## 规模
184个PDF格式的论文。
